# Images 目录说明

本目录存放唐朝模组所有图像资源。请将所有图片保存为 **PNG** 格式。

## 各子目录说明

| 目录 | 内容 | 建议尺寸 |
|------|------|----------|
| `NationIcons/` | 文明图标（领袖头像、旗帜等） | 128×128 px |
| `UnitSprites/` | 单位精灵图（游戏中单位的像素图） | 64×64 px 每帧 |
| `UnitIcons/` | 单位列表图标 | 32×32 px |
| `BuildingIcons/` | 建筑/奇观图标 | 32×32 px |
| `TechIcons/` | 科技图标 | 32×32 px |
| `TileIcons/` | 地块设施图标 | 32×32 px |
| `ResourceIcons/` | 资源图标 | 32×32 px |
| `PromotionIcons/` | 单位晋升图标 | 32×32 px |
| `WonderImages/` | 奇观大图（弹出插画） | 256×256 px |
| `TerrainImages/` | 地形贴图 | 64×64 px（可平铺） |

> **注意**: 已删除 `PolicyIcons/` 目录（模组不包含政策内容）。
