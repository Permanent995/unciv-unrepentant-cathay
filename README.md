# 无悔华夏 · Unrepentant Cathay

> 一个 [Unciv](https://github.com/yairm210/Unciv) 扩展模组，以中国古代王朝为主题的文明包。目前已收录**秦朝**、**盛唐**、**晚唐**、**蒙古帝国**四个文明。计划开发宋朝和明朝。

## 🏯 目前已收录文明

### 秦朝 · Qin Dynasty

> **「受命于天，既寿永昌！」**

始皇帝统领的强力军事文明，擅长征服与扩张。

| 特性 | 说明 |
|------|------|
| **领袖** | 始皇帝 |
| **文明特色** | 武秦 |
| **首都** | 咸阳 |
| **偏好胜利** | 征服胜利 |

**独特能力：** 陆军单位获军功授爵晋升、平民获士子晋升、马/铁资源加成、大军事家加速

**独特单位：** 秦武卒（步兵）、秦强弩（弩兵）

**独特建筑：** 郡县制、统一度量衡

**独特改良：** 学宫、矿洞

**独特资源：** 铜、玉、士子、醴

---

### 盛唐 · Sheng Tang

> **「愿天可汗受天命、抚万民、开疆拓土，开创大唐盛世！」**

天可汗李世民统领的黄金文明，骑兵无敌，奇观兴邦。

| 特性 | 说明 |
|------|------|
| **领袖** | 李世民（天可汗） |
| **文明特色** | 贞观之治 |
| **偏好胜利** | 文化胜利 |

**独特能力：** 黄金时代+50%长度、骑乘单位+15%战力、奇观建造+20%产能

**独特单位：** 玄甲军（重骑）、陆羽（茶圣伟人，开启黄金时代）

**独特建筑：** 国子监（大学）、大雁塔（奇观）、大运河（奇观）

**独特改良：** 茶园（仅盛唐可建，产出茶叶奢侈品）

**独特资源：** 茶叶

**专属事件：** 科举、茶经问世

---

### 晚唐 · Late Tang

> **「大唐气数将尽，藩镇割据，社稷危矣！陛下当力挽狂澜，中兴大唐！」**

唐宪宗统领的衰落王朝，军事削弱但经济韧性强，需通过削藩中兴。

| 特性 | 说明 |
|------|------|
| **领袖** | 唐宪宗 |
| **文明特色** | 日薄西山 |
| **偏好胜利** | 征服胜利 |

**独特能力：** 陆军维护费-50%、河流地块+3金。征服城市增加割据指数，建立城市增加正统。割据指数过高触发全局惩罚。

**独特单位：** 神策军（伟人，消耗获得正统+3，并召唤3个骑手）

**独特建筑：** 平藩（消耗正统，移除藩镇，降低割据指数，+3锤瓶琴快乐，+9金币）

**藩镇割据系统：**
- 征服城市 → 割据指数 +1
- 建立城市 → 正统 +1
- 割据 >5/10/15 → 全局金币/文化/产能 -5%
- 事件获取神策军 → 消耗得正统 → 建造平藩削藩

**专属事件：** 科举、削藩之议

---

### 蒙古帝国 · Mongol Empire

> **「草原上的狼群，只有最强者才能统领！」**

成吉思汗统领的游牧征服文明，以骑兵为核心，席卷欧亚大陆。

| 特性 | 说明 |
|------|------|
| **领袖** | 成吉思汗 |
| **文明特色** | 草原牧群 |
| **偏好胜利** | 征服胜利 |

**独特能力：** 牧场+3锤、骑乘单位+15%战力+1移动、陆军维护费-75%、马匹资源+300%、征服城市获100科技

**独特单位：** 草原汗王（重骑，神学解锁，30力6速，自带征服者晋升）、把阿秃儿（弓骑，骑术解锁，14力24远程，移动后攻击）

**独特建筑：** 忽里勒台（哲学解锁，骑乘单位+50%产能，牧场+1信仰，马匹+1金，每5人口+2快乐）

**独特晋升：** 征服者（+100%经验/攻城/攻击军事单位，杀敌获金币，相邻敌军-50%力，与军事单位相邻+15%力）

**专属万神殿：** 长生天（每个牧场+1信仰+1食物，每个马匹+1文化）

**专属事件：** 忽里勒台大会（每35回合，免费草原汗王 or 300金+100锤）

## 📁 安装方法

1. 下载本仓库（Code → Download ZIP）
2. 解压到 Unciv 的 `mods/` 目录下
3. 在 Unciv 游戏内 **主菜单 → Mods** 中启用「无悔华夏」
4. 开始新游戏，选择秦朝/盛唐/晚唐/蒙古帝国

```
mods/
  └── unciv unrepentant cathay/
      ├── jsons/          # 数据配置
      ├── music/          # 音乐
      ├── Images/         # 图标资源
      ├── credits.md      # 素材来源
      └── README.md
```
