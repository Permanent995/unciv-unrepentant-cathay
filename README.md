## 🏯 目前已收录文明

### 秦朝 · Qin Dynasty

> **「受命于天，既寿永昌！」**

始皇帝统领的**强力军事文明**，擅长征服与扩张。

| 特性 | 说明 |
|------|------|
| **领袖** | 始皇帝（Qin Shi Huang） |
| **文明特色** | 武秦 |
| **首都** | 咸阳 |
| **偏好胜利** | 征服胜利 |
| **起始地形** | 平原 |

**独特能力（Uniques）：**
- 🏛️ 首都额外 `+3 科研`
- ⚔️ 所有陆军单位获得**军功授爵**晋升
- 🐎 每处马匹资源 `+3 产能`
- ⚒️ 每处铁矿资源 `+1 产能`
- 🎖️ 大军事家获取速度 `+50%`

**独特单位：**
- **秦武卒** — 精锐步兵
- **秦强弩** — 远程弩兵

**独特建筑：**
- **郡县制** — 行政效率，每5人口加2快乐
- **统一度量衡** — 经济标准化

**独特改良设施：**
- **学宫** — 学术机构
- **矿洞** — 增产矿井

**独特资源：**
- 铜（Bronze） / 玉（Jade）

## 📁 安装方法

1. 下载本仓库（Code → Download ZIP）
2. 解压到 Unciv 的 `mods/` 目录下
3. 在 Unciv 游戏内 **主菜单 → Mods** 中启用「无悔华夏」
4. 开始新游戏，选择秦朝文明

```
mods/
  └── 无悔华夏/
      ├── jsons/          # 数据配置
      ├── music/          # 音乐（CK3 OST，非商用）
      ├── Images/         # 图标资源
      ├── credits.md      # 素材来源
      └── README.md
```
